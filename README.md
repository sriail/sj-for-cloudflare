# scramjet-templates
A collection of setup files for Scramjet, drag-and-drop, easy to use.

## Supported Versions:
- v2.0.67-alpha.1


## Supported Languages
- React
- HTML/JS

## Quick Note For React or Framework Based Projects:
No Scramjet, Epoxy, or Libcurl files are hosted in the `react` folder. You will need to install them yourself using your own package manager.

We recommend **bun.**
