# scramjet-templates
A collection of setup files for Scramjet, drag-and-drop, easy to use.

## Supported Versions:
- v2.0.67-alpha.1
- v2.0.67-alpha
- v2.0.6

## Supported Languages
- React
- HTML/JS

## Quick Note:
This repository does not host any Scramjet, Epoxy, Libcurl, or Scramjet Controller files. You will need to install packages yourself and move files around.
